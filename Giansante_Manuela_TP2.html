2. Use one of the viewed methods (simple exponential smoothing, Holt, Holt-Winter) which is appropriated to your series. Justify your choice. Comment on your models parameter(s). How well does your model fit the data?

Concluding that we can identify a trend in the data, but no seasonality it would be pertinent to use the Holts linear trend method (A,N). It includes a forecast equation and two smoothing equations.

\(\hat{y}_{t+h|t}=l_t+hb_t\)

\(l_t= \alpha y_t+(1-\alpha )(l_{t-1}+b_{t-1})\)

\(b_t= \beta^*(l_t-l_{t-1})+(1-\beta ^*)b_{t-1}\)

is the smoothing parameter for the level and it takes values from 0 to 1.

* is the smoothing parameter for the trend.

ita_exp_ts <- ts(ita_exp$Exports, start = 1960)
fit1<- holt(ita_exp_ts, initial = "optimal", h=4)

We firstly model the time series specifying the initial argument to optimal, meaning that the initial values are optimized along with the smoothing parameters using ets.

In this case we did not specify the alpha value, therefore it has been estimated, same goes for the beta parameter. Respectively, they amount to 0.9485 and 1*10^(-04). The alpha is the smoothing parameter for the level, it being close to 1 indicates that the estimation process is very responsive to each new observation.

The optimized initials states \(l_t\) and\(b_t\), which in this order give an estimate of the local level of the data generating process at time t and the estimate of the trend at time t.

Following there are the model selection criteria, Akaikes IC, AIC corrected for small sample bias, and finally the Bayesian Information Criterion. When we will confront many different model we will look for the process with the lowest values.Usually these three indices point in the same direction for model selection, maybe BIC will favour a more simpler method.

The scale dependent error measures MAE and RMSE, tells respectively that on average the errors magnitude is of 0.9336816 and the fitted model will vary from the true process by 1.295316.

By simply judging on graphical we can see that the estimated process moves very closely to the true time series. In this case we have seen that the data has had a lot of ups and owns, which will probably influence the future exports as well.So maybe trying a damped method were the trend is reduced within long run horizons, would be appropriate.

We try with the Damped Holts Method (Ad, N):

\(\hat{y}_{t+h|t}=l_t+\phi b_{t-1}\\l_t=\alpha y_t+(1-\alpha)(l_{t-1}+\phi b_{t-1})\\b_t=\beta^*(l_t-l_{t-1})+(1-\beta^*)\phi b_{t-1}\)

The method introduce a parameter phi which indicates how the trend the forecast will flatten through time.

3. Run model with different values of the parameter(s). Plot it against parameter(s) values. How each parameter does impact the forecast?

So far we have let the function do the work in terms of estimating the optimal alpha and beta parameters.

par(mfrow=c(2,2)) # graphics

for(i in seq(0.2,0.8,0.2)){
  model.holt = holt(ita_exp_ts, alpha = i, beta = i,initial = "optimal", h = 4)

  alpha_beta_txt = paste("Holt's alpha and beta:", i, sep=" ")
  rmse = round(sqrt(sum(model.holt$residuals^2)/length(model.holt$residuals)),2)
  rmse_txt = paste("rmse:", rmse, sep=" ")
  title_txt = paste(alpha_beta_txt, rmse_txt, sep =" ")
 
  plot(model.holt, main = title_txt, PI=FALSE)
  lines(fitted(model.holt), lwd=2, col="red")
  }

The RMSE decreases as we increase the values for beta and alpha. The fitted model is less smooth as the smoothing parameters grow closer to 1, ergo more sensitive to new observations. Yet, if we compare the bottom right plot with (which has alpha and beta lower in value than what the function estimated in the previous computations) the model fit1 we see that the dips are much deeper than the tre series, and the peaks are higher. Because the model grows in sensitivity the forecast changes too, we can in fact notice that the slop estimate fr the four point forecast changes. The last model is capable of capturing the the increasing behaviour in the data during the latest years, in fact the forecasts is steeper in slope than the predictions computed previously.
